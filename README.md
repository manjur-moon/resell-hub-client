# ReSell Hub Client

## Purpose
Frontend for ReSell Hub, a second-hand marketplace platform.

## Features
- Public home/products/categories/about/contact pages
- Better Auth email/password and Google login UI
- Buyer, seller, and admin dashboards
- Product listing with search/filter/sort/pagination
- Product CRUD UI for sellers
- Wishlist and checkout pages
- Stripe payment success/cancel pages
- Responsive dark/light UI
